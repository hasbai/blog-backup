Free V2ray

2018-08-10
  • web tool

　　最近发现有一个程序托管的网站可以免费搭建 v2ray，自己测试了一下，速度还行，下载 1-2M/s，延迟 200ms 以上，自己看看视频网页是够用了。故写一篇教程，希望大家低调使用。
　　
　　
　　
　　打开heroku 网站并注册，邮箱填自己的，其他可以乱填。注意：注册需要加载谷歌人机验证，需要科学上网，否则会报错。注册后登录，可能需要验证邮箱地址。


　　
　　打开 GitHub 的代码库v2ray 一键搭建脚本，点紫色的按钮 Deploy to Heroku
　　
　　输入 app name，建议地区选欧洲，速度似乎较快。
　　
　　点击随机生成 UUID，替换掉默认的 UUID，然后点 Deploy app。
　　
　　等待一会，生成好了以后点最右边的 settings，config vars 里面点 reveal config vars，可以看到 UUID，复制下来配置的时候要用。在 Domains and certificates 里面有一个链接，点开，如果显示 Bad request，就说明成功了。
　　
　　进入 v2ray 配置，主机名填刚刚打开的域名，注意是域名，没有 https，即 xxxxx.herokuapp.com，UUID 填刚刚复制的，其它照着图片中的填。
　　
　　配置好了以后就可以了。
　　
　　感谢 v2ray 一键脚本的开发者hxc1427418039和Project V让我们能够用到自由的网络。
